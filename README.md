# Scatterpub Toolchain — Example Project

A worked example for the [scatterpub-toolchain](https://github.com/scattercode/scatterpub-toolchain): a set of open tools for digitising physical books and preparing manuscripts for publication.

This project contains a small sample of real scanned pages from *Tell Me, Bella* by Vahan Totovents, published by the [Armenian Institute](https://armenianinstitute.org.uk). Follow the tutorial to take those scans all the way through to a clean, reviewed Word document ready for sharing or importing into publishing software such as [Vellum](https://vellum.pub).

---

## What you will need

- A **Mac** (some tools in the chain are macOS-only)
- **[Claude Code](https://claude.ai/code)** — for the AI-assisted copy-editing step
- A few free command-line tools — the tutorial walks you through installing them

No prior experience with the command line is assumed. The tutorial explains each step.

**[→ Read the tutorial](TUTORIAL.md)**

## About the sample text

*Tell Me, Bella* is a collection of short stories by the Armenian author Vahan Totovents (1893–1937), translated into English by Mischa Kudian. The five sample scans cover the opening pages of the book. The full text is published by the Armenian Institute.

---

## About the toolchain

The scripts and skills used in this example live in [scattercode/scatterpub-toolchain](https://github.com/scattercode/scatterpub-toolchain). That repository is included here as a submodule — you do not need to clone it separately.

If you want to use the toolchain for your own book project, see the toolchain README for instructions on setting up a new project from scratch.
